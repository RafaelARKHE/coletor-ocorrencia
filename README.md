# Coletor de Ocorrência — 3º GBM

App de campo (PWA) do sistema **Nexus** para captura guiada de dados e áudio durante
ocorrências de bombeiro. Abrir no **Safari (iOS)** e **Adicionar à Tela de Início**;
funciona offline após instalado.

> Este repositório contém **apenas o app** (interface + schema), sem nenhum dado de
> ocorrência. Os dados coletados ficam **somente no aparelho** e são exportados como
> pacote para processamento no Nexus (repositório privado). Nada sensível aqui.

- `coletor.html` — o app.
- `campos_ocorrencia.json` — schema adaptativo de papéis/campos por tipo de ocorrência.
- `spike_iphone.html` — teste de compatibilidade do navegador.
